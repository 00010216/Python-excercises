Create python virtual environment:

For windows:
```
py -m venv env
```
For Linux:
```
python -m venv env
```

Activate environment:

For Windows:
```
.\env\Scripts\activate
```
For Linux:
```
source env/bin/activate
```

Install required libraries 

For Windows:
```
py -m pip install -r requirements.txt
```

For Linux:
```
python -m pip install -r requirements.txt
```
